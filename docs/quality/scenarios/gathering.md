# Gathering Scenarios

## S-G001: 나무 채벌 (맨손)

- **전제조건**: 에이전트 근처에 나무(oak log), 인벤토리 비어있음
- **명령**: "근처 나무 캐"
- **측정 기준**:
  - 턴 수, 비용
  - `mine_area` 사용 여부 (vs `mine_block` 반복)
  - 자동 아이템 흡수 작동 여부 (`pickup_items` 수동 호출 없이)
  - 채굴 애니메이션 (lookAt + swingArm)
- **관련 컴포넌트**: `actions`, `brain`, `body`

### 평가 관점

1. **목표 달성** — 나무를 실제로 캤는가? 인벤토리에 나무 관련 아이템(oak_log 등)이 있는가?
2. **도구 선택** — mine_area vs mine_block 중 상황에 적절한 것을 사용했는가? 여러 블록을 캘 때 mine_area가 효율적.
3. **행동 효율** — 불필요한 이동, 중복 관찰(get_observation 반복), 쓸모없는 액션이 없었는가?
4. **실패 복구** — 액션 실패 시 적절히 대응했는가? (위치 조정, 다른 도구, 재시도 등)
